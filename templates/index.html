Projeto Chuva

Esse serviço web foi pensado exclusivamente na análise de dados de precipitação da bacia do Rio Piaui, localizada no sul de Alagoas.

Projeto desenvolvido para o Programa Institucional de Bolsas de Iniciação Científica (PIBIC) pela bolsista Verena Meirelles, com orientação do Prof. Dr. Chistopher Freire.
